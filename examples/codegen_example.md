# Ocular Code Generation Example

Since Gleam doesn't have macros, Ocular provides a code generator to create lenses for your records automatically.

## Setup

Add to your `gleam.toml`:

```toml
[dependencies]
ocular = ">= 1.0.0"

[dev-dependencies]
glance = ">= 0.11.0 and < 1.0.0"
simplifile = ">= 2.0.0 and < 3.0.0"
```

Then copy the full implementation from `examples/ocular_gen_full.gleam` to `src/ocular_gen.gleam`.

## Your Domain Model

Create `src/models.gleam`:

```gleam
// src/models.gleam
pub type User {
  User(
    name: String,
    email: String,
    age: Int,
  )
}

pub type Address {
  Address(
    street: String,
    city: String,
    zip: String,
  )
}
```

## Generate Lenses

Run the generator:

```sh
gleam run -m ocular_gen -- src/models.gleam src/models/lenses.gleam
```

## Generated Output

The generator creates `src/models/lenses.gleam`:

```gleam
// src/models/lenses.gleam
// AUTO-GENERATED by ocular_gen - do not edit manually

import ocular
import ocular/types.{type Lens}
import models

// User lenses

pub fn user_name() -> Lens(models.User, models.User, String, String) {
  ocular.lens(
    get: fn(u) { u.name },
    set: fn(v, u) { models.User(..u, name: v) },
  )
}

pub fn user_email() -> Lens(models.User, models.User, String, String) {
  ocular.lens(
    get: fn(u) { u.email },
    set: fn(v, u) { models.User(..u, email: v) },
  )
}

pub fn user_age() -> Lens(models.User, models.User, Int, Int) {
  ocular.lens(
    get: fn(u) { u.age },
    set: fn(v, u) { models.User(..u, age: v) },
  )
}

// Address lenses

pub fn address_street() -> Lens(models.Address, models.Address, String, String) {
  ocular.lens(
    get: fn(a) { a.street },
    set: fn(v, a) { models.Address(..a, street: v) },
  )
}

pub fn address_city() -> Lens(models.Address, models.Address, String, String) {
  ocular.lens(
    get: fn(a) { a.city },
    set: fn(v, a) { models.Address(..a, city: v) },
  )
}

pub fn address_zip() -> Lens(models.Address, models.Address, String, String) {
  ocular.lens(
    get: fn(a) { a.zip },
    set: fn(v, a) { models.Address(..a, zip: v) },
  )
}
```

## Usage

Now you can use the generated lenses:

```gleam
import ocular
import ocular/compose as c
import models
import models/lenses as lens

pub fn example() {
  let user = models.User(name: "Alice", email: "alice@example.com", age: 30)
  
  // Get
  let name = ocular.get(lens.user_name(), user)
  
  // Set
  let older = ocular.set(lens.user_age(), 31, user)
  
  // Modify
  let uppercase = ocular.over(lens.user_name(), string.uppercase, user)
  
  // Compose
  // If User had an address field:
  // let city_lens = lens.user_address() |> c.lens(lens.address_city())
}
```

## Multiple Files

You can also generate lenses for multiple files at once:

```sh
# Generate from all files in src/domain/
gleam run -m ocular_gen -- "src/domain/*.gleam" src/generated/lenses.gleam
```

## Build Integration

Add to your build process in `package.json` or Makefile:

```json
{
  "scripts": {
    "generate": "gleam run -m ocular_gen -- src/models.gleam src/models/lenses.gleam",
    "build": "npm run generate && gleam build"
  }
}
```

Or in a `Makefile`:

```makefile
.PHONY: generate build

generate:
	gleam run -m ocular_gen -- src/models.gleam src/models/lenses.gleam

build: generate
	gleam build
```
